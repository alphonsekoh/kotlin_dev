### Project Setup

#### Requirements

- An IDE like IntelliJ IDEA (Recommended) or Eclipse
- Basic programming fundamentals

#### Setting it up

- Click on `New Project`
- Press on the `Kotlin` tab
- Choose the JVM libraries you want to build kotlin based on.
